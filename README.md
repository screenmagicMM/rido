# rido
Taxi/Ride Sharing Service Investment Analysis
## Installation
> Python 3.5 Is required.
> pip install -r requirements.txt 
will install all necessary packages. 

## Running Application

Once Installed just run  
$python app.py


## Requirements
> Python 3.5
> Sqlite
> flask-adminlte
> sqlalchemy
> python-dateutil
